About #VIBEJAM

The 2025 Vibe Coding Game Jam, or #vibejam, is a MIND-BENDING competition where developers build games using AI tools, emphasizing a "vibe coding" approach. Hosted by @levelsio, this event encourages RADICAL INNOVATION in game development, particularly for web-based platforms.

Participants are tasked with creating games where at least 80% of the code is generated by AI, accessible without login, and free-to-play, preferably on their own domain. Multiplayer is optional but encouraged. The games should avoid loading screens and heavy downloads, ensuring INSTANT GRATIFICATION!!!

$$$ BIG MONEY PRIZES $$$

The competition offers a $5,000 PRIZE for the best game, with judging by a panel including @karpathy, @timsoret, @mrdoob, @s13k_, and @levelsio. Sponsors include @boltdotnew and @coderabbitai.
